#### AVL树
以两位发明人的名字首字母命令，公认第一种能够自平衡的树
平衡二叉树，任意节点左右子树的高度差不能超过1
高度与节点数量之间的关系是O(logn)
实现了AVL树添加、删除等常规方法

为了维护平衡，需要维护高度这个变量，计算平衡因子（左子树高度-右子树高度）然后通过左旋转，右旋转来维持平衡
```java
    // LL情况，即新添加节点位于当前不平衡节点左孩子的左孩子
    // 对节点y进行向右旋转操作，返回旋转后新的根节点x
    //        y                              x
    //       / \                           /   \
    //      x   T4     向右旋转 (y)        z     y
    //     / \       - - - - - - - ->    / \   / \
    //    z   T3                       T1  T2 T3 T4
    //   / \
    // T1   T2
    // RR情况，即新添加节点位于当前不平衡节点右孩子的右孩子
    // 对节点y进行向左旋转操作，返回旋转后新的根节点x
    //    y                             x
    //  /  \                          /   \
    // T1   x      向左旋转 (y)       y     z
    //     / \   - - - - - - - ->   / \   / \
    //   T2  z                     T1 T2 T3 T4
    //      / \
    //     T3 T4
    // LR情况
    // 对节点x进行向左旋转操作，然后对节点y向右旋转
    //        y                              y                         z
    //       / \                           /   \                    /    \
    //      x   T4     向左旋转 (x)        z     T4    向右旋转(y)   x      y
    //     / \       - - - - - - - ->   /  \      - - - - - ->   /  \   /  \
    //   T1   z                        x   T3                   T1  T2 T3  T4
    //       / \                     /  \
    //     T2  T3  
```
